# 🎨 Retro Pixel Editor

A modern **Retro Pixel Art Editor** built with **HTML, CSS, and JavaScript**. Create pixel art directly in your browser with drawing tools, customizable color palettes, undo/redo functionality, and high-quality PNG export.

---

## 🚀 Features

- 🎨 Pencil Tool
- 🩹 Eraser Tool
- 🪣 Paint Bucket (Flood Fill)
- 🎯 Color Picker (Eyedropper)
- 📐 Multiple Grid Sizes
  - 8×8
  - 16×16
  - 32×32
  - 64×64
- 🌈 Built-in Retro Color Palettes
- 🎨 Custom HEX Color Picker
- 🕘 Recent Colors
- ↩️ Undo / Redo
- 📏 Toggle Grid Lines
- 💾 Save Draft (Local Storage)
- 📂 Load Saved Draft
- 🖼️ Export Pixel Art as PNG
- ⌨️ Keyboard Shortcuts
- 📱 Responsive Design

## 🎮 Keyboard Shortcuts

| Shortcut | Action |
|----------|--------|
| **P** | Pencil |
| **E** | Eraser |
| **B** | Bucket Fill |
| **I** | Color Picker |
| **G** | Toggle Grid |
| **C** | Clear Canvas |
| **Ctrl + Z** | Undo |
| **Ctrl + Y** | Redo |
| **Esc** | Close Shortcuts |
